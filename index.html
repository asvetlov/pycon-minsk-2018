UTF-8

  • 1-4 bytes per code point
  • Can encode everything
  • [\u0000-\u007F] 0bbb_bbbb
  • [\u0080-\u07FF] 110b_bbbb+10bb_bbbb
  • [\u0800-\uFFFF] 1110_bbbb+10bb_bbbb*2
  • [\U00010000-\U00010FFFF] 1111_0bb+10bb_bbbb*3

UTF-32

  • 4 bytes per code point
  • Little Endian (UTF-32-LE) ‚Üí B1 B2 B3 B4
  • Big Endian (UTF-32-BE) ‚Üí B4 B3 B2 B1

UTF-16

  • 2 or 4 bytes per code point
  • Gory inheritage
  • Surrogate pairs:
  • D800-DBFF + DC00-DFFF

U+1F872 (ü°≤) to UTF-16 ‚Üí D83E + DC72

BOM

Byte order mark

A magic number at the start of text

  • EF BB BF ‚Üí UTF-8
  • FE FF ‚Üí UTF-16-BE
  • FF FE ‚Üí UTF-16-LE
  • 00 00 FE FF ‚Üí UTF-32-BE
  • FF FE 00 00 ‚Üí UTF-32-LE
